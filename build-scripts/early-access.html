How to get started

1
Download the Purple Computer software (.iso file) using the button above.
2
Grab any USB drive. Make sure there's nothing you need on it: we'll be erasing it! Any old drive with 8GB or more should work.
3
Download balenaEtcher (free) to any computer. You'll use it to "flash" the USB drive: this means rewriting the whole USB drive into a format your old laptop can start from.
4
Open balenaEtcher, pick the Purple .iso file and your USB drive, and click Flash.
5
Now you have a Purple Key! See the cards below to boot it on your laptop.
